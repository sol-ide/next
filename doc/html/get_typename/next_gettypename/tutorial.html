Tutorial

This tutorial section shows some of the features of Next.GetTypename in the form of annotated, relatively simple, code snippets.

Before each example, includes are : 

#include <next/get_typename/get_typename.hpp>

Get typename from a struct or a class name 

struct an_type
{
};

BOOST_REQUIRE_EQUAL( next::get_typename< an_type >(), "an_type" );
