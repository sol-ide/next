Introduction

The Next.Event library is an implementation of event dispatching. It defines :

  • a dispatcher
  • events
  • events handlers

The dispatcher manages a thread pool for sending event. Each event handler has a thread group and each thread group can handle events sequencialy ( without synchronization primitives as mutex )
